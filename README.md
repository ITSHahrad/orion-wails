# Orion-Wails Boilerplate

🚀 A modern starter template for building desktop applications with **React 18**, **Wails v2 (Go)**, and **Tailwind CSS v3**.

## Overview

This boilerplate provides a pre-configured setup for creating cross-platform desktop applications with a sleek, responsive frontend and a powerful Go backend using the Wails framework. It integrates React 18 for the UI, Tailwind CSS v3 for styling, and Vite for a fast development experience.

## Tech Stack

- **Frontend**: React v18 (with TypeScript support)
- **Styling**: Tailwind CSS v3 (with JIT compilation)
- **Build Tool**: Vite v4 (optimized for speed)
- **Backend**: Wails v2 (Go-based framework for desktop apps)
- **Platforms**: Windows, macOS, Linux

## Key Features

- 🛠️ **Pre-configured Wails + React**: Seamless integration between Go backend and React frontend.
- 🎨 **Tailwind CSS v3**: Fully set up with JIT for rapid, responsive styling.
- ⚡ **Vite-powered**: Lightning-fast development and hot module replacement (HMR).
- 📱 **Responsive Design**: Includes example layouts for mobile and desktop.
- 🧰 **TypeScript Ready**: Optional TypeScript support for type-safe development.
- 🏗️ **Production Ready**: Optimized build scripts for cross-platform deployment.

## Prerequisites

- [Go](https://golang.org/dl/) (v1.18 or later)
- [Node.js](https://nodejs.org/) (v16 or later)
- [Wails CLI](https://wails.io/docs/gettingstarted/installation) (`go install github.com/wailsapp/wails/v2/cmd/wails@latest`)
